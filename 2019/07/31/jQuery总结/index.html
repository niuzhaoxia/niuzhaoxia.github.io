'jQuery总结'

发表于 2019-07-31

1、掌握jq常用的获取元素对象的方式以及常用的选择器

jquery通过选择器的方式获取的元素，不一定只获取一个，而是不管获取到几个都会返回一个元素集合（类数组），这些元素在类数组中获得了一个索引值。原生方法获取不到时会返回null，jq获取不到仍然返回一个对象。在jq的返回对象中有一个length属性，这个属性表示获取到的个数。

  1. 基本选择器
  • id 选择器 #ID
  • className 类选择器 .className
  • 标签选择器 p ul li div
  1. 层级选择器
    • 后代选择器 就是一个空格 #tabBox li （获取id为tabBox的所有子孙代中的li）；
    • 子级选择器 > #tabBox > ul (只在儿子代中查找)
    • 兄弟选择器 + .header + div
    • 获取所有的弟弟 li.active ~
  1. 过滤选择器
    • :first 获取第一个
    • :last 获取最后一个
    • :odd 获取奇数索引的
    • :even 获取偶数索引的
    • :not(selector) 除了selector的都要； :not(first) 除了第一个都要
    • :eq(index) 获取索引为index的
    • :gt(index) 获取比索引index大的
    • :lt(index) 获取比索引index小的
  1. 内容选择器
    • :contains(‘内容’) 获取包含指定内容的元素
    • :empty 获取空标签元素
    • :has(selector) 获取包含指定选择器元素的元素
    • :hidden 获取不可见的元素对象
  1. 属性选择器
    • [attr=val] 获取含有属性attr并且其值是val的元素
    • [attr^=val] 获取含有attr属性并且其值是以val开头的元素
    • [attr$=val] 获取含有attr属性并且其值是以val结尾的元素
    • 5.4 [attr*=val] 获取含有attr属性并且其值包含val的元素

2、jq对象和原生元素对象互转

jq对象转原生对象：jq对象中索引对应的就是原生元素对象，我们可以通过[索引]就能获取到这个原生对象，此外还有两种方法可以获取这个原生对象

原生对象转jq对象：将我们通过原生方法获取到原生对象当做参数传给$方法（$(原生对象)）就可以把原生对象转成jq对象。

3、jq常用的方法

以 let $tabBox = $(‘#tabBox’) 为例

  1. height(): 不传参数是获取元素对象的高度，传参就是修改元素对象的高度
  1. width(): 获取元素对象的宽度，传递参数是设置元素对象的宽度
  1. offset 获取元素到body左侧和上侧的偏移距离，返回值是一个对象{left: ,top: }
  1. scrollTop() / scrollLeft() 获取纵向 / 横向滚动条卷去的距离
  1. innerHeight() / innerWidth()获取当前元素的高度 / 宽度 等效于
  1. outerHeight() / outerWidth() 获取元素包含边框的高度/宽度 相当于 offsetHeight / offsetWidth
  1. hasClass(类名) 判断当前元素有没有这个类名，有就返回true否则返回false
  1. addClass(类名)为当前元素追加类名，返回当前jq元素对象（return this）
  1. removeClass(类名) 移除当前元素的某一个类名，返回当前jq元素对象
  1. append() 向元素对象内追加元素
  1. 元素.appendTo(容器)将当前插入到指定容器中
  1. prepend() / prependTo() 向容器开头添加元素对象
  1. before() / after() 给当前元素之前或者之后添加【兄弟】元素

  1. insertBefore() / insertAfter 将当前元素添加到参照元素之前或者之后

  2. replaceAll(selector) 用当前元素替换selector匹配到元素

  1. 元素.remove() 移除元素

  2. next() / nextAll() 获取当前元素的下一个弟弟 / 获取所有的弟弟

  3. prev() / prevAll()获取当前元素的上一个哥哥 / 获取当前元素所有的哥哥

  4. children() 获取当前元素的元素儿子

  5. parent() 获取当前元素的元素父亲

  6. filter(selector) 在同级中把满足selector的元素获取到（同级查找，在兄弟之间查找）

  7. find(selector) 从当前元素的后代中获取满足selector的元素

  8. css 方法，

  • 元素.css(‘width’) 获取元素的width
  • 元素.css(‘width’, ‘200px’) 设置元素的width
  • 元素.css({width: ‘200px’, backgroundColor: ‘#00b38a’})
  • $(‘#tabBox’).css(‘height’, 250); // 不写单位，默认单位是px
  • console.log($(‘#tabBox’).css(‘width’)); // 返回了一个字符串，后面就不能链式调用（链式调用也叫级联）
  • console.log($(‘#tabBox’).css(‘width’, 200)); // 返回了jq对象，可以继续链式调用

  1. text() 传参就是设置元素的innerText，不传是获取元素的innerText

  2. html() 传参就是设置元素的innerHTML，不传参就是获取innerHTML

4、jq绑定事件和移除事件的方法

1、 将原生事件名中on去掉，例如onclick，jq提供了一个对应事件名的方法，click() ，要把事件触发时的执行的事件函数作为参数传给click方法。

2、 on() 方法：语法：jq对象.on(事件名[不带on], 事件函数);

3、 trigger() 用代码触发事件 参数就是不带on的事件名 ；返回值是当前jq对象，可以继续链式调用

4、off() 移除事件，参数是不带on的事件名，返回值是当前jq对象，可以继续链式调用

5、jq简单动画效果

hide() / show() 隐藏元素 / 显示元素 参数是完成动画时间(ms) ，还可以写 ‘slow’慢（600ms）， ‘fast’ 快(200ms)

slideUp() / slideDown() 向上滑动收起 / 向下滑动展开

fadeIn() / fadeOut() 淡入/淡出

slideToggle() 第一次如果是收起的，执行slideToggle就是展开，如果上一次是展开的，我们执行slideToggle就是收起，适合做点击展开，再点击收起的需求

animate() 语法：$(selector).animate(target, duration, easing, afterCallback)

stop() 停止当前元素正在进行的动画，不管到没到终点都得停；
finish() 停止当前元素正在进行的动画，但是它是忽略动画规定的时间，一下子到终点。
