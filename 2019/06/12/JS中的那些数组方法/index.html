JS中的那些数组方法

发表于 2019-06-12

在数组类(Array) 原型对象中提供了很多供数组调用的属性方法，这些方法让我们能够更加方便的来操作数组

从以下几个方面对这些数组方法进行分类

  • 作用
  • 返回值
  • 是否改变原数组

push

方法说明：向数组末尾添加 一项或多项
原数组：改变
返回值：添加后的数组长度

unshift

方法说明：向数组开头添加 一项或多项
原数组：改变
返回值：添加后数组的长度

pop

方法说明：删除数组最后一项
原数组：改变
返回值：删除的那一项

shift

方法说明：删除数组第一项
原数组：改变
返回值：删除的那一项

splice

数组的增删，具体使用分五种情况

情况一：splice() 不传参数

情况二： splice() 传一个参数

情况三：splice() 传两个参数

情况四：splice() 传三个及以上参数

情况五：splice() 传递三个及以上参数，第二项参数为零

结合splice的各种参数情况

  • splice(0, 0, x)可实现unshift()功能,可向数组开头添加一项或者多项；
  • splice(array.length, 0, x)可实现push()功能，可向数组末尾添加一项或者多项；
  • splice(-1) 可实现pop()功能，删除数组最后一项
  • splice(0, 1) 可实现shift()功能， 删除数组第一项

slice

数组的截取，将截取的内容组成新数组（浅拷贝）返回，原数组不改变。slice() 具体的参数情况如下：

slice()/slice(0) 不传参或者只传0，等价于将原数组克隆一份作为返回值

slice(n) 只传递一个参数，从索引为n处一直截取到数组末尾，将截取到的内容以新数组返回

slice(n, m) 从索引为n处截取到索引为m处（但不包括索引为m的这一项），将截取到的内容以新数组返回
